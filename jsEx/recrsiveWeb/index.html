These are three approaches to recursively calculating the sum of an array.
Any recursive pattern will work as long as all indexes are counted exactly once.

1. Dividing the first element in the array from the remainder of the array
2. Dividing the last element in the array from the remainder of the array
3. Dividing the array in half at the midpoint until there are <= 2 elements
